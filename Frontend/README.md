# Next.js E-Commerce Learning Components (Razorpay Practice Project)

This setup gives you:

- Modern reusable components
- Responsive Tailwind UI
- Clean folder structure
- ES7+ syntax
- Industry-style patterns
- Beginner-friendly architecture

---

# Recommended Stack

- Next.js 15+
- React 19+
- Tailwind CSS
- TypeScript
- App Router

---
